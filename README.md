# Data

The dataset used in this paper are publicly available and have been analyzed in these publications:
- Village dataset: Face-to-face interactions collected in a French village between members of voluntary associations and members of local political institutions. This dataset is presented by [Azaiez and Salot 2025](https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0311626)
- European Commission dataset: meetings between European Commission members and stakeholders from 2014 to 2024. This dataset is presented by [Azaiez and Mandel 2025](https://appliednetsci.springeropen.com/articles/10.1007/s41109-025-00735-6)

# Structure
`src` Python modules for hypergraph models, metrics, and utility functions.
\
`scripts` 
Python scripts for running simulations or metrics using multiprocessing.
\
`notebooks` Jupyter notebooks for loading metrics and generating figures.
